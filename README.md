## Опис проєкту

**Minerland** — це 2D гра у жанрі Simulator, створена з використанням рушія Unity.

Проєкт орієнтований на дослідження підземного світу, процедурну генерацію шахти, видобуток ресурсів, покращення інструментів та поступовий розвиток ігрового персонажа.

Основною метою роботи є розробка модульної архітектури гри, реалізація систем генерації світу, механіки копання, інвентаря, економіки та взаємодії між ігровими компонентами з використанням можливостей Unity і Tilemap System.

## Основні можливості
- Процедурна генерація підземного світу
- Tilemap-система побудови та оновлення карти
- Система копання блоків
- Система інвентаря та стекування ресурсів
- Економічна система та продаж ресурсів
- Система покращення кирок
- Контролер персонажа та фізика руху
- Модульна архітектура ігрових систем
- Інтеграція UI з логікою гри через систему подій
- Оптимізація продуктивності при роботі з великою кількістю блоків
- Використання ScriptableObject для конфігурації даних
- Підтримка масштабування та подальшого розширення функціоналу гри

# Інструкція зі встановлення та запуску

## 1. Завантаження проєкту

1. Перейти за посиланням Google Drive
2. Завантажити архів із грою

---

## 2. Розархівування проєкту

1. Відкрити завантажений `.zip` архів
2. Розархівувати папку проєкту у зручне місце, наприклад на Робочий стіл

---

## 3. Запуск гри

1. Відкрити папку `Minerland`
2. Знайти файл:

```plaintext
Minerland.exe
```

3. Запустити файл `Minerland.exe`

Після запуску гра автоматично відкриється та буде готова до використання.


# Примітка

Для коректної роботи гри операційна система Windows може попросити підтвердження запуску файлу. У такому випадку необхідно натиснути:

**"Run anyway" / "Все одно запустити"**
